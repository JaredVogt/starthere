3. `git clone https://github.com/JaredVogt/utilities ~/projects/utilities` ([utilities](https://github.com/JaredVogt/utilities))
4. get dropbox up and running
5. restore mackup
7. `~/dotfiles/macos/createSymlinks.sh`
